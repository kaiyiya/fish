# 智能鱼类电商推荐系统项目概述

## 一、项目背景与意义

随着电子商务的快速发展和人工智能技术的不断成熟，个性化推荐系统已成为提升用户体验和商业价值的关键技术。在垂直电商领域，特别是生鲜电商中，如何准确识别商品并基于用户行为提供精准推荐，是提升用户满意度和转化率的核心问题。

本项目针对鱼类电商场景，设计并实现了一个融合深度学习图像识别技术与协同过滤推荐算法的智能推荐系统。系统通过深度学习模型实现鱼类图像的自动识别，解决了用户"看图识鱼"的需求；同时，基于用户历史行为数据，采用协同过滤算法提供个性化商品推荐，有效提升了用户体验和购买转化率。

## 二、项目目标

本项目旨在构建一个完整的智能鱼类电商推荐系统，主要目标包括：

1. **实现高精度的鱼类图像识别**：基于深度学习技术，构建能够准确识别多种鱼类品种的图像分类模型，识别准确率达到90%以上。

2. **构建高效的推荐算法体系**：实现基于用户的协同过滤（UserCF）、基于物品的协同过滤（ItemCF）以及混合推荐策略，为用户提供个性化的商品推荐服务。

3. **解决推荐系统冷启动问题**：通过基于内容的推荐和热门推荐策略，为新用户和新商品提供有效的推荐方案。

4. **构建完整的电商业务系统**：实现用户管理、商品管理、订单管理等核心电商功能，为推荐系统提供数据支撑。

5. **提供良好的用户体验**：基于Taro框架开发跨平台小程序，支持微信小程序、H5等多端运行，为用户提供便捷的购物体验。

## 三、技术架构

### 3.1 系统架构

本系统采用前后端分离的微服务架构，主要包括以下部分：

- **前端层**：基于Taro 3.x框架开发，支持微信小程序、H5等多端运行，使用React 18和TypeScript构建用户界面。

- **后端层**：基于NestJS 10.x框架构建RESTful API服务，采用TypeScript开发，提供统一的接口服务。

- **数据层**：使用MySQL作为主数据库存储业务数据，通过TypeORM进行数据访问和ORM映射。

- **AI服务层**：集成深度学习图像识别模型和推荐算法服务，提供智能识别和推荐功能。

## 四、核心功能模块

### 4.1 AI图像识别模块

**技术实现：**
- 采用MobileNetV3Large作为基础模型，利用ImageNet预训练权重进行迁移学习
- 通过数据增强技术（旋转、缩放、翻转等）提升模型泛化能力
- 使用早停机制和学习率自适应调整防止过拟合
- 支持模型转换为TensorFlow.js和ONNX格式，便于跨平台部署

**功能特点：**
- 支持多种鱼类品种的识别（可扩展至20+种）
- 识别准确率高，Top-1准确率可达90%以上
- 返回识别置信度和备选结果，提升用户体验
- 记录识别历史，支持用户查看识别记录

### 4.2 智能推荐模块

**协同过滤算法：**

1. **基于用户的协同过滤（UserCF）**
   - 算法原理：找到与目标用户兴趣相似的其他用户，推荐这些相似用户喜欢的商品
   - 相似度计算：采用余弦相似度和皮尔逊相关系数
   - 时间复杂度：O(n²×m)，其中n为用户数，m为商品数
   - 适用场景：用户数量相对稳定，用户兴趣多样化

2. **基于物品的协同过滤（ItemCF）**
   - 算法原理：找到与用户喜欢的商品相似的商品进行推荐
   - 相似度计算：基于商品-用户矩阵计算商品相似度
   - 时间复杂度：O(m²×n)，其中m为商品数，n为用户数
   - 适用场景：商品数量相对固定，商品特征稳定

3. **混合推荐策略**
   - 结合UserCF和ItemCF的优势，采用加权融合的方式
   - 公式：Score(hybrid) = α × Score(UserCF) + β × Score(ItemCF)
   - 默认权重：α = 0.6, β = 0.4
   - 推荐效果通常优于单一算法

**基于内容的推荐：**
- 基于商品特征（价格、分类、标签等）和用户历史偏好
- 构建用户偏好向量和商品特征向量
- 使用余弦相似度计算匹配度
- 适用于解决冷启动问题

**行为权重设计：**
- 浏览（view）：1.0
- 收藏（collect）：3.0
- 加入购物车（add_cart）：5.0
- 购买（purchase）：10.0
- 分享（share）：2.0

## 五、技术特点与创新

### 5.1 深度学习技术的应用

1. **迁移学习优化**：采用MobileNetV3预训练模型，通过迁移学习大幅减少训练时间和数据需求，同时保持较高的识别准确率。

2. **模型轻量化**：MobileNetV3作为轻量级网络，模型参数量少，推理速度快，适合在移动端和服务器端部署。

3. **数据增强策略**：采用多种数据增强技术，包括旋转、缩放、翻转、剪切等，有效提升模型泛化能力。

### 5.2 推荐算法的优化

1. **多算法融合**：实现了UserCF、ItemCF、基于内容的推荐和混合推荐等多种算法，系统可根据用户数据情况智能选择最佳推荐策略。

2. **冷启动解决方案**：
   - 新用户：使用热门推荐和基于内容的推荐
   - 新商品：使用基于内容的推荐
   - 系统冷启动：使用热门推荐和分类推荐

3. **行为权重设计**：根据用户行为的重要性设计不同的权重，购买行为权重最高（10.0），浏览行为权重最低（1.0），更准确地反映用户偏好。

4. **性能优化**：
   - 数据库索引优化：为用户行为表添加复合索引
   - 推荐结果缓存：减少重复计算
   - 增量计算：只计算新用户的推荐，避免全量重算

### 5.3 系统架构设计

1. **模块化设计**：采用NestJS的模块化架构，各功能模块独立，便于维护和扩展。

2. **前后端分离**：前端和后端完全分离，通过RESTful API通信，支持多端部署。

3. **跨平台支持**：前端基于Taro框架，一套代码可编译为微信小程序、H5等多种平台。

本项目设计并实现了一个融合深度学习图像识别技术与协同过滤推荐算法的智能鱼类电商推荐系统。系统通过MobileNetV3迁移学习实现高精度的鱼类图像识别，通过多种协同过滤算法提供个性化商品推荐，有效解决了垂直电商场景下的商品识别和推荐问题。项目采用现代化的技术栈和架构设计，具有良好的可扩展性和可维护性，为垂直电商领域的智能化应用提供了有价值的参考和实现方案。
