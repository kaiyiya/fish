# 论文格式要求总结

根据《计科毕业设计参考模板.docx》整理的详细格式要求。

## 一、文档结构要求

### 1. 封面
- 标题：**本科毕业设计说明书**
- 包含信息表格：题目、学生姓名、学号、学院、专业、指导教师、行业导师（可选）、年月

### 2. 前置部分
- **毕业论文独创性声明**：作者签名、日期
- **毕业论文版权使用授权书**：作者签名、导师签名、日期

### 3. 摘要部分
- **中文摘要**：
  - 标题：**摘  要**（一级标题）
  - 内容：详细描述研究内容、方法、结果
  - 关键词格式：**关键词：×××；×××；×××；×××；×××**（用分号分隔）

- **英文摘要**：
  - 标题：**Abstract**
  - 内容：英文翻译
  - Keywords格式：**Keywords: ×××; ×××; ×××; ×××; ×××**（用分号分隔）

### 4. 目录
- 标题：**目   录**
- 格式：一级标题、二级标题、三级标题，带页码
- 包含：结论、参考文献、致谢、附录等

### 5. 正文结构
- 使用三级标题结构：一级标题、二级标题、三级标题
- 章节编号：1、1.1、1.1.1
- 正文段落可以使用（1）、（2）等编号

### 6. 结论
- 标题：**结  论**

### 7. 参考文献
- 标题：**参 考 文 献**
- 按文献类型分类（A-J），每种类型有固定格式

### 8. 致谢
- 标题：**致  谢**
- 格式要求：小四号宋体，行距为固定值22磅

---

## 二、字体和格式要求

### 1. 标题格式
- **一级标题**：章节标题（如"1 绪论"）
- **二级标题**：小节标题（如"1.1 研究背景"）
- **三级标题**：小点标题（如"1.1.1 ××××"）

### 2. 正文格式
- **中文**：宋体
- **英文**：Times New Roman
- **字号**：正文通常为小四号或五号（根据具体要求）

### 3. 致谢格式
- **字体**：小四号宋体
- **行距**：固定值22磅

---

## 三、图表格式要求

### 1. 图（Figure）
- **位置**：图序和图标题置于**图的下方**，居中
- **字体**：宋体五号
- **布局**：选择"嵌入型"
- **编号规则**：按章连续编号
  - 格式：图X-Y（X为章号，Y为序号）
  - 例如：第二章第一个图为"图2-1"，第二个图为"图2-2"
  - **不得重复或跳缺**
- **引用**：正文中引用时使用"（见图2-1）"

### 2. 表（Table）
- **位置**：表序和表标题置于**表的上方**，居中
- **字体**：
  - 表标题：宋体五号
  - 表格内中文：宋体五号
  - 表格内英文：Times New Roman五号
- **编号规则**：按章连续编号
  - 格式：表X-Y（X为章号，Y为序号）
  - 例如：第二章第一个表为"表2-1"，第二个表为"表2-2"
- **表格样式**：**三线表**
  - 上下两条粗线：宽1.5磅
  - 其余细线：宽0.75磅
- **续表**：如果表格跨页，使用"续表X-Y"格式
- **引用**：正文中引用时使用"（见表2-1）"

### 3. 公式（Formula）
- **对齐**：公式应**居中对齐**
- **序号**：
  - 标注序号并加圆括号
  - 序号用阿拉伯数字逐章编序
  - 编号方式与插图、表格方式一致
  - 序号排在公式**右边行末**，**右对齐**
  - 格式：（X-Y），例如（2-1）
- **换行规则**：
  - 长公式在一行无法写完时，应在等号（或数学符号，如"+"、"-"号）处换行
  - 数学符号在换行后的行首
- **字体要求**：
  - 一般变量（如坐标X、Y，电压V，频率f）：**斜体**
  - 矢量：**粗斜体**如S或白斜体上加单箭头
  - 常用函数（如三角函数cos、对数函数ln等）、数字运算符、化学元素符号及分子式、单位符号、产品代号、人名地名的外文字母：**正体**
- **变量说明**：
  - 在公式后的段落中，采用"式中：A为某某，B为某某"的方式说明
  - A、B等字符必须与公式中的字体一致（如公式中为斜体，说明中也必须使用斜体）

### 4. 代码（Code）
- **要求**：
  - 保证结构和逻辑上的完整性
  - 尽量提供核心代码
  - 尽量避免前端的代码
  - 相似功能的代码无需重复提供
- **格式**：
  - 使用高亮显示工具编辑后以文本形式粘贴
  - 高亮显示工具参考：
    1. https://highlightcode.parryqiu.com/
    2. https://www.jyshare.com/front-end/5536/
  - 可以加上段落边框

## 四、参考文献格式要求

### 文献类型及标志代码
- M - 普通图书
- C - 会议录
- G - 汇编
- N - 报纸
- J - 期刊
- D - 学位论文
- R - 报告
- S - 标准
- P - 专利
- DB - 数据库
- CP - 计算机程序
- EB - 电子公告

### 格式示例

#### A. 期刊论文 [J]
```
[序号] 主要责任者. 文献题名[J]. 刊名, 出版年份, 卷号(期号): 起止页码．
[1] 袁庆龙, 候文义. Ni-P合金镀层组织形貌及显微硬度研究[J]. 太原理工大学学报, 2001, 32(1): 51-53.
```

#### B. 专著 [M]
```
[序号] 主要责任者. 文献题名[M]. 出版地: 出版者, 出版年: 起止页码．
[2] 刘国钧, 王连成. 图书馆史研究[M]. 北京: 高等教育出版社, 1979: 15-18．
```

#### C. 论文集、会议录 [C]
```
[序号] 主要责任者. 论文集名: 其他论文集名信息[C]. 出版地: 出版者, 出版年: 页码.
[3] 辛希孟. 信息技术与信息服务国际研讨会论文集: A集[C]. 北京: 中国社会科学出版社, 1994.
```

#### D. 论文集、会议录的析出文献 [A]
```
[序号] 析出文献主要责任者. 析出文献题名[A]. 论文集主要责任者. 论文集名: 其他论文集名信息[C/文献载体标识]. 出版地: 出版者, 出版年: 页码.
```

#### E. 学位论文 [D]
```
[序号] 主要责任者. 文献题名[D]. 出版地: 出版者, 出版年: 页码.
[7] 张和生. 地质力学系统理论[D]. 太原: 太原理工大学, 1998: 1-10.
```

#### F. 科技报告 [R]
```
[序号] 主要责任者. 文献题名[R]. 报告地: 报告会主办单位, 年份: 页码.
```

#### G. 专利文献 [P]
```
[序号] 专利申请者或所有者. 专利题名: 专利国别, 专利号[P]. 公告日期或公开日期.
```

#### H. 国际、国家标准 [S]
```
[序号] 起草责任者. 标准代号. 标准顺序号—发布年 标准名称[S]. 出版地: 出版者, 出版年: 页码.
```

#### I. 报纸文章 [N]
```
[序号] 作者. 文献题名[N]. 报纸名, 出版日期(版次).
```

#### J. 电子文献 [EB/OL]
```
[序号] 主要责任者. 题名: 其他题名信息[文献类型标志/文献载体标志]. 出版地: 出版者, 出版年: 引文页码(更新或修改日期).
```

## 五、页面和排版要求

### 1. 分页
- 封面、声明、摘要、目录、正文各章节之间需要分页
- 使用分页符（Page Break）

### 2. 段落格式
- 正文段落首行缩进（通常2字符）
- 段落间距适当
- 行距根据内容调整（致谢部分为固定值22磅）

### 3. 编号规则
- **章节编号**：1、1.1、1.1.1（三级标题）
- **图表编号**：按章连续编号，格式为"图X-Y"、"表X-Y"
- **公式编号**：按章连续编号，格式为"（X-Y）"
- **不得重复或跳缺**

---

## 六、特殊要求

### 1. 致谢部分
- 文字要简捷、实事求是
- 切忌浮夸和庸俗之词
- 格式：小四号宋体，行距为固定值22磅

### 2. 代码插入
- 使用高亮工具处理后再插入
- 可以加段落边框
- 避免重复代码

### 3. 图表引用
- 正文中引用图表时使用"（见图X-Y）"或"（见表X-Y）"格式

---

## 七、格式检查清单

- [ ] 封面信息完整（题目、姓名、学号、学院、专业、指导教师等）
- [ ] 声明和授权书已填写
- [ ] 中英文摘要完整，关键词格式正确
- [ ] 目录结构完整，页码正确
- [ ] 正文使用三级标题结构
- [ ] 图表编号按章连续，无重复或跳缺
- [ ] 图表标题位置正确（图在下方，表在上方）
- [ ] 表格使用三线表格式
- [ ] 公式居中对齐，序号右对齐
- [ ] 参考文献格式正确，包含中英文文献
- [ ] 致谢格式正确（小四号宋体，行距22磅）
- [ ] 所有章节之间正确分页

---

## 八、注意事项

1. **图表编号**：必须按章连续编号，不得重复或跳缺
2. **字体统一**：中文用宋体，英文用Times New Roman
3. **表格样式**：必须使用三线表
4. **公式格式**：居中对齐，序号右对齐
5. **代码格式**：使用高亮工具处理
6. **参考文献**：至少10篇，英文至少2篇
7. **致谢格式**：小四号宋体，行距固定值22磅
