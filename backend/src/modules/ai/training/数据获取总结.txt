========================================
数据获取总结 - 最实用的方法
========================================

由于直接从网络批量下载图片涉及版权、API限制等问题，
推荐使用以下方法：

【方法1】Kaggle数据集（最推荐，5分钟搞定）
----------------------------------------
1. 访问: https://www.kaggle.com/datasets/crowww/a-large-scale-fish-dataset
2. 注册/登录Kaggle（免费）
3. 点击"Download"下载数据集（约几百MB）
4. 解压后，将图片按类别整理到:
   backend/src/modules/ai/training/data/fish_images/
   例如:
   - salmon/ 文件夹放入三文鱼图片
   - sea_bass/ 文件夹放入鲈鱼图片
   - 等等...
5. 每类选择 20-50 张图片即可开始训练

优点: 数据质量高、数量充足、已标注
时间: 5-10分钟

【方法2】手动收集（快速测试，10-15分钟）
----------------------------------------
1. 打开浏览器，访问 Google Images 或 Bing Images
2. 搜索以下关键词并下载图片:
   - "salmon fish" → 放入 salmon/ 文件夹
   - "sea bass fish" → 放入 sea_bass/ 文件夹
   - "tuna fish" → 放入 tuna/ 文件夹
   - "carp fish" → 放入 carp/ 文件夹
   - 等等...
3. 每类下载 10-20 张图片
4. 图片格式: jpg, jpeg, png 都可以

优点: 快速、灵活
时间: 10-15分钟

【方法3】使用Kaggle API（自动化）
--------------------------------
1. 安装: pip install kaggle
2. 获取API凭证（见Kaggle下载指南.txt）
3. 运行: kaggle datasets download -d crowww/a-large-scale-fish-dataset
4. 解压并整理

优点: 自动化、可重复
时间: 首次设置需要10分钟，之后一键下载
