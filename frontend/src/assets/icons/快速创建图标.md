# 快速创建TabBar图标

## 当前状态
图标路径已配置，但图标文件还未创建。现在tabBar只显示文字。

## 快速解决方案（3选1）

### 方案1：使用在线图标库（最快，5分钟）

1. **访问图标库**：
   - 中文：https://www.iconfont.cn/
   - 英文：https://www.flaticon.com/

2. **搜索并下载图标**：
   - 首页：搜索 "home" 或 "首页"
   - 识别：搜索 "camera" 或 "相机"
   - 搜索：搜索 "search" 或 "搜索"
   - 我的：搜索 "user" 或 "用户"

3. **处理图标**：
   - 下载后调整为 81x81 像素
   - 普通状态：颜色 #666666
   - 激活状态：颜色 #1890ff
   - 保存为PNG格式

4. **重命名并放入目录**：
   ```
   frontend/src/assets/icons/
   ├── home.png
   ├── home-active.png
   ├── recognize.png
   ├── recognize-active.png
   ├── search.png
   ├── search-active.png
   ├── profile.png
   └── profile-active.png
   ```

5. **取消注释图标路径**：
   在 `app.config.ts` 中取消注释 `iconPath` 和 `selectedIconPath`

### 方案2：使用HTML生成器（简单）

1. 打开 `create_simple_icons.html`
2. 在浏览器中打开
3. 点击下载按钮保存图标
4. 将图标放入 `frontend/src/assets/icons/` 目录
5. 取消注释 `app.config.ts` 中的图标路径

### 方案3：暂时只显示文字（当前方案）

如果暂时不需要图标，可以保持当前配置（图标路径已注释），tabBar会只显示文字标签。

## 图标要求

- **尺寸**：81x81 像素（Taro推荐尺寸）
- **格式**：PNG（支持透明背景）
- **颜色**：
  - 普通状态：#666666
  - 激活状态：#1890ff

## 完成后

1. 将图标文件放入 `frontend/src/assets/icons/` 目录
2. 在 `app.config.ts` 中取消注释图标路径
3. 重启前端开发服务器
4. 刷新页面查看效果
